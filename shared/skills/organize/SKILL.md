# File Organization Skill
1. Ask user for target directory path (do NOT search broadly)
2. List all files with `ls -la <path>`
3. Categorize files by type/company/date
4. Present organization plan as a table BEFORE moving anything
5. Execute moves in batches of 10, confirming after each batch
6. Save a manifest of all moves to `<path>/reorganization_log.md`
